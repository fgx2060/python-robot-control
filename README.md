# python-robot-control
Python 機器人顏色感測驅動的智慧機器人設計

## 📖 專案簡介
本專案利用 Python 控制 mBot2 智慧機器人，結合感測器數據，實現自動化運作。  
專案重點在於 **機器人控制邏輯設計** 及 **感測器整合應用**。

## ⚙️ 功能特色
- mBot2 機器人基礎驅動
- 顏色感測器自動判斷
- 路徑控制與迴避機制

## 🛠️ 技術架構
- **硬體**：mBot2
- **程式語言**：Python
- **感測器**：顏色 / 距離

## 🚀 主要控制程式
```bash
python main.py
```
